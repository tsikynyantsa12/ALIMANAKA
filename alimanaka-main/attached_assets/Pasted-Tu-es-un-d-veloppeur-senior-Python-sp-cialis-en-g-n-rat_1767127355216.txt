Tu es un développeur senior Python spécialisé en génération PDF professionnelle.
Tu dois créer une application Python complète qui génère automatiquement
un calendrier A3 paysage (2 pages, 6 mois par page) à partir de fichiers CSV.

========================
OBJECTIF DU PROJET
========================

Générer un PDF A3 paysage très stylisé :
- 2 pages
- 6 mois par page (mois verticaux, gauche → droite)
- Calendrier combiné :
  - Liturgique (dimanches, couleurs, versets)
  - Église (programmes journaliers)
  - Agricole (icônes uniquement)
  - Phases lunaires
  - Jours fériés
- Qualité impression professionnelle

========================
CONTRAINTES TECHNIQUES
========================

- Langage : Python 3.11+
- Génération PDF : ReportLab
- Images : Pillow
- Données : CSV (compatibles Excel)
- Aucun framework web
- Fonctionnement offline
- Code modulaire, lisible, pédagogique (niveau L2 Génie Logiciel)

========================
MISE EN PAGE EXACTE
========================

FORMAT :
- A3 paysage (420 × 297 mm)
- 2 pages
- Marges fines
- Image de fond église légère (opacity faible)

STRUCTURE D’UN JOUR (ligne horizontale) :

[ GAUCHE | CENTRE | DROITE ]

GAUCHE (étroit) :
- Numéro du jour
- Nom du jour (Lun, Mar…)
- Fond coloré selon couleur liturgique
- Pas de surcharge

CENTRE (large) :
- Nom du dimanche (si existant)
- Versets bibliques (sur une ligne)
- Programmes de l’église
- Jours fériés
- Icône phase lunaire

DROITE (étroit) :
- Icônes agricoles uniquement
- 1 action = 1 ligne
- Plusieurs actions possibles par jour
